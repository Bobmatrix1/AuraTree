# ✨ Aura Tree

![Aura Tree Banner](./frontend/public/aura%20tree%20logo.png)

> **Elevate your online presence with a link-in-bio platform that actually feels like you.**

Aura Tree is a premium, high-performance link management platform designed for creators, developers, and brands. Built with a focus on aesthetics and user experience, it features stunning glassmorphic themes, real-time analytics, and seamless mobile-first management.

[View Live Demo](https://auratree.link) | [Create Your Page](https://auratree.link/signup)

---

## 🎨 Premium Features

- **🚀 Instant Customization**: Change your entire profile's look with one click using our curated premium themes.
- **📊 Real-time Analytics**: Track visits, link clicks, and QR scans with deep insights (Pro/Teams).
- **📱 Mobile-First Dashboard**: Manage your links, appearance, and settings effortlessly from any device.
- **🛡️ Secure & Scalable**: Built on Firebase and Node.js for industrial-grade security and speed.
- **💳 Automatic Billing**: Seamless subscription management integrated with Paystack for recurring monthly billing.
- **🖥️ Admin Control Panel**: Powerful dashboard for site owners to manage users, prices, and platform health.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS + Framer Motion
- **Animations**: GSAP (GreenSock) + Lenis Smooth Scroll
- **Icons**: Lucide React + React Icons

### Backend
- **Runtime**: Node.js + Express
- **Database**: Google Firestore (NoSQL)
- **Auth**: Firebase Authentication (Admin SDK)
- **Payments**: Paystack API Integration
- **Storage**: Cloudinary (Avatar & Background hosting)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- Firebase Account
- Paystack Account (for payments)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Bobmatrix1/AuraTree.git
   cd AuraTree
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   # Create a .env file with your Firebase config
   npm run dev
   ```

3. **Setup Backend**
   ```bash
   cd backend
   npm install
   # Create a .env file based on .env.example
   npm run dev
   ```
